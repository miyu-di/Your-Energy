# Your Energy
 
 Your Energy — це сучасний веб-застосунок для пошуку вправ та підтримки здорового способу життя. Проєкт реалізований як курсова робота, що демонструє навички роботи з чистим JavaScript, інтеграцію з зовнішніми API та створення адаптивного інтерфейсу.

 Мета застосунку — надати користувачеві зручний каталог вправ із фільтрацією за групами м'язів, частинами тіла та обладнанням.

## Основні функції:
Цитата дня: Щоденне натхнення, яке завантажується безпосередньо з API при кожному візиті.

Фільтрація: Зручне перемикання між категоріями (Muscles, Body parts, Equipment).
Каталог вправ: Перегляд карток вправ із рейтингом та основними характеристиками.
Детальна інформація: Модальне вікно з повним описом вправи, відео-інструкцією (якщо доступно) та можливістю оцінювання.
Пошук: Знаходьте потрібні вправи за ключовими словами.
Система оцінювання: Можливість залишити свій рейтинг для кожної вправи через PATCH запит.
Підписка: Форма підписки на оновлення через email.
Пагінація: Зручна навігація по великому списку вправ та фільтрів.

## Технологічний стек

1. Проєкт побудований на принципах "Vanilla JS" без використання важких фреймворків:

2. Мова: JavaScript (ES6+).

3. Збірка: [Vite](https://vite.dev/) — забезпечує швидку розробку та оптимізацію коду.

4. Стилізація: CSS (Flexbox, Grid, адаптивна верстка під Mobile/Tablet/Desktop).

5. API: [YourEnergy Swagger API.](https://your-energy.b.goit.study/api-docs/#/)

6. HTML-модульність: Використання partials для структурування коду.

## Структура файлів
src/index.html — головна сторінка.

src/favorites.html — сторінка обраних вправ.

src/partials/ — HTML-фрагменти (header, footer, hero тощо).

src/css/ — стилі проєкту.

src/main.js — точка входу JavaScript.

public/ — статичні ресурси (іконки, фавікони).
